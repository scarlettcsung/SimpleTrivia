# SimpleTrivia
SimpleTrivia is a program that facilitates a basic trivia game with your friends and family. It takes a json file with questions and answers, and displays a random question per selected category. 

## Usage
To upload your trivia, upload it as a .json file in /jsons. There is a template titled trivia_template.json in the directory that you may use.  
Run src/main.py in your IDE or using the terminal, navigate to the project directory, then run: `python src/main`
